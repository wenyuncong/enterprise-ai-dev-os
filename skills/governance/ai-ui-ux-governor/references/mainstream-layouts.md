# Mainstream ERP Layout Notes

Sources (official design systems):
- SAP Fiori: Shell Bar + Dynamic Page patterns (global header + content-centric pages)
- Microsoft Fluent: NavigationView (left nav for sections, adaptive)
- Dynamics 365 design principles: clarity, consistency, efficiency, reduce cognitive load

Key takeaways:
1. Persistent global header with identity, search, and profile actions.
2. Left navigation for primary modules; minimize depth; group sections.
3. Content pages use a header region with key actions and status, followed by scrollable content sections.
4. Data-heavy screens: summary KPIs above filters and tables; keep filters reachable.
5. Responsive behavior: collapse navigation and preserve the primary call-to-action.
