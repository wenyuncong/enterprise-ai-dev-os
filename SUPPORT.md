# Support

## Questions And Discussions

Use GitHub Discussions for:

- usage questions
- tool compatibility reports
- ideas for new skills
- migration stories
- adapter verification notes

Suggested discussion categories:

- Announcements
- Q&A
- Ideas
- Tool Compatibility
- Showcase

## Issues

Use GitHub Issues for:

- reproducible bugs
- documentation errors
- adapter path corrections
- audit script failures
- missing or broken checks

## Pull Requests

Use pull requests for concrete changes. Please follow `CONTRIBUTING.md` and include verification output.

## Private Material

Do not post private strategy, customer details, unredacted logs, local paths, or raw project archives in public issues or discussions.
